# API Documentation

## Endpoints

### GET /api/health
Health check endpoint

### GET /api/geo
Geolocation data based on IP

### GET /api/products
Product list with filtering

### GET /api/posts
Blog posts with search

### GET /api/sitemap.xml
XML sitemap for SEO

### GET /api/robots.txt
Robots.txt configuration

### GET /api/schema
JSON-LD schema data

## Rate Limiting

All endpoints: 100 requests per minute
