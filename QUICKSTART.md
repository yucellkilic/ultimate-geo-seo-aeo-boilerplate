# Ultimate Geo-SEO & AEO Boilerplate - Quick Start

## 🚀 Installation

```bash
git clone https://github.com/yucellkilic/ultimate-geo-seo-aeo-boilerplate.git
cd ultimate-geo-seo-aeo-boilerplate
npm install
cp .env.example .env.local
npm run dev
```

Open http://localhost:3000
