# Contributing

## How to Contribute

1. Fork repository
2. Create feature branch: `git checkout -b feature/my-feature`
3. Make changes
4. Run tests: `npm run test`
5. Commit: `git commit -m "feat: add feature"`
6. Push and create PR

## Commit Convention

- `feat:` new feature
- `fix:` bug fix
- `docs:` documentation
- `style:` formatting
- `refactor:` refactoring
- `test:` tests
- `chore:` dependencies
